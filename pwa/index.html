What this app does

NaijaFactCheck uses a fine-tuned XLM-RoBERTa model with LoRA to detect fake news in Nigerian languages. Paste any news text and get an AI verdict in seconds.

🌍

Supported languages

English Hausa Yoruba Pidgin Igbo — coming soon
⚙️

How it works

  • Paste text from WhatsApp, Twitter, or any source
  • The model detects the language automatically if set to Auto
  • XLM-RoBERTa analyses linguistic patterns across 4 languages
  • Returns a FAKE or REAL verdict with a confidence percentage
  • Results below 70% confidence should be treated with caution
🧠

The AI model

  • Base model: XLM-RoBERTa (Facebook AI)
  • Fine-tuned with LoRA for memory-efficient training
  • African language data generated via NLLB-200 neural translation
  • Training datasets: LIAR, GonzaloA/fake_news, MT-projected Nigerian texts
  • Evaluated with Macro-F1 across Hausa, Yoruba, Pidgin, English
XLM-RoBERTa LoRA NLLB-200 FastAPI HuggingFace
⚠️

Limitations

  • AI can make mistakes — double-check important stories
  • Code-switched text (mixed languages) may reduce accuracy
  • Images, audio, and video are not analysed — text only
  • African language labels are silver-standard (MT-projected)
  • Training data reflects patterns up to early 2025

Research

This app is the deployment artefact of a master's thesis on multilingual fake news detection for Nigerian languages, using cross-lingual transfer learning with silver-standard label projection via NLLB-200 neural machine translation.
